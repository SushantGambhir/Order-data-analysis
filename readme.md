# 📦 Order Data ETL & Analysis (Python + SQL Server)

## 📌 Overview

This project demonstrates a beginner-friendly **ETL (Extract, Transform, Load) pipeline** built using Python and SQL Server on a retail order dataset sourced from Kaggle.

The primary goal of this project is to **build a strong foundation in data engineering concepts** by implementing a complete data pipeline — from data extraction to storage — along with basic validation and analysis.

---

## 🎯 Objectives

* Understand the end-to-end ETL workflow
* Work with real-world datasets using APIs
* Perform data cleaning and transformation using Pandas
* Store structured data in a relational database (SQL Server)

---

## 🧰 Tech Stack

* **Python**
* **Pandas**
* **SQL Server**
* **Kaggle API**

---

## 🔄 ETL Pipeline Architecture

### 🔹 Extract

* Data is programmatically downloaded using the Kaggle API
* Eliminates manual data handling
* Ensures reproducibility

### 🔹 Transform

* Data cleaning and preprocessing using Pandas:

  * Handling missing values
  * Removing duplicates
  * Standardizing column formats
  * Converting data types

### 🔹 Load

* Cleaned data is loaded into SQL Server
* Structured tables are created for efficient querying
* Data is inserted using Python database connectors

---

## ✅ Key Learnings

* Practical understanding of ETL pipeline design
* Working with APIs for data ingestion
* Data cleaning and transformation using Pandas
* Writing SQL queries for validation and analysis
* Structuring code into modular components

## 📌 Disclaimer

This is a **learning-focused project** aimed at building foundational data engineering skills. The emphasis is on clarity, structure, and understanding rather than production-level complexity.
